Introduction to CSS

It is important to separate content from presentation because it enables an easier time maintaining code, bug fixing and working with others on code.

This is where CSS comes in.

CSS is what we style our presentation on HTML with.

We can use divs with it, change an object’s colour, make borders, format and more!

The css file is also referred to as the “Style sheet”.
