How to add images to web pages

Images can improve the design and the appearance of a web page.Images are added to web pages by first, putting the image into a folder on the website. Then, use the SRC command and the file path, which should add the desired image. For proper validation, the importance of which we will discuss later, use alt for screen readers or incase the file doesn't display properly.

Example:

<img src="pic_trulli.jpg" alt="Italian Trulli">

<img src="img_girl.jpg" alt="Girl in a jacket">

<img src="img_chania.jpg" alt="Flowers in China">
