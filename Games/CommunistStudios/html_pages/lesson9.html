File organisation

Files are easy to manage and organise, as long as you use folders. You can add files to folders by dragging the desired files into the desired folders. To create a folder, right click on the side-banner containing all content and select: new folder… Rename as desired.

Side-banner containing files

Their location is represented as “../folder_name/possible_other_folder_name/file_name.file_type” you can think of it as going through each country, state ect separated by a slash, then arriving at a specific house of a specific type. However, if you are in a suburb an you want to go to somewhere else you can use the “../” to go to the top, e.g. “../folderName/maybeOtherFolderName/file.filetype”
