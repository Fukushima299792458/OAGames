Classes and ids

Class is used for styling divs, while ids are used for styling specific elements.

Here we can see the effect of class and ids on a website.

The id is able to override the div, enabling extensive customization as opposed to if only divs could customise a webpage. Likewise, divs enable more output with less code, as instead of putting an id on every single element you can put in 1 div. This improves readability of the code, and saves time.
